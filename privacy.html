Privacy Policy for YouTube Loop Extension

Effective Date: April 12, 2025

Introduction

Thank you for using the YouTube Loop browser extension ("the Extension"). This privacy policy explains how we handle information when you use the Extension. Our goal is transparency and respecting your privacy. The Extension's sole purpose is to allow you to select a start and end time for a YouTube video and play that specific section in a loop.

Information We Handle

The Extension needs to handle minimal information solely to perform its function:

  • User-Provided Loop Times: The start and end times (in seconds) that you input into the extension's popup interface.
  • Video Playback Time: The extension interacts with the YouTube video player on the active YouTube page to read the current playback time and to set the playback time (seek) according to the loop times you provide.

The Extension **does not** collect, store, or transmit any Personally Identifiable Information (PII) such as your name, email address, location, or user accounts.

The Extension **does not** track your Browse history or activity on websites other than interacting with the video player on the YouTube page (`youtube.com`) where you actively use the extension.

How We Use Information

The information handled is used exclusively within your browser for the following purposes:

  • To receive the start and end points for the video loop from you via the popup.
  • To read the YouTube video's current time to check if it has reached the loop's end point.
  • To set the YouTube video's current time back to the loop's start point when the end point is reached.

All processing happens locally within your browser on the active YouTube page.

Data Storage

To provide a more convenient user experience, the Extension may store the last used start and end loop times locally within your browser's storage mechanism (specifically, using the chrome.storage API). This allows the extension to remember your previous settings the next time you use the popup. This data is stored only on your device (or synced via your browser profile if using chrome.storage.sync) and is not accessed by the developer.

Data Sharing and Transmission

We do not share, sell, or transmit any data handled by the Extension to the developer or any third parties. All functionality and data handling occur locally within your web browser. The loop times and interaction with the video player are not sent anywhere outside of your computer.

Security

Since all data handling is performed locally within your browser and no data is transmitted, the security relies on the security measures of your own browser and computer system. We have designed the extension to use the minimum necessary permissions and data handling required for its function.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We encourage you to review this policy periodically for any changes. Updates will be reflected by the "Effective Date" at the top of this policy.
